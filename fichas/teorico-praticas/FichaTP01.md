## Algoritmo de procura em profundidade

Ordem de expansão: `A` -> `B` -> `D` -> `H` -> `I` -> `E` -> `J` -> `K` -> `C` -> `F` -> `L`

Solução: `A` -> `C` -> `F` -> `L`

## Algoritmo de procura em largura

Ordem de expansão: `A` -> `B` -> `C` -> `G`

Solução: `A` -> `C` -> `G`

## Algoritmo de procura de custo uniforme

Ordem de expansão: `A` -> `B` -> `C` -> `G`

Solução: `A` -> `C` -> `G`
